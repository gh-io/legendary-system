**AgentGateway** is an AI-native gateway designed to route, secure,
observe, and orchestrate autonomous AI agents in production.

for agent systems.

## Why AgentGateway?

- Central control plane for AI agents
- Deterministic routing & policy enforcement
- Secure tool & API execution
- First-class observability for agent behavior
